源起

假设在640px的设计稿中,某图片的设计宽度为100px。此时，在设备分辨率640px的情况下，图片与整体页面布局的比例正好合适，比较美观。那么在设备分辨率为320px,则该图片的宽度应该设为50px,对页面进行等比例缩放的工作,这样页面还是比较美观。

问题?

一旦设备分辨率与设计稿的分辨率不一致,所有涉及到宽度的地方都要进行修改，比较繁琐。

解决方法

使用rem代替px设置如图片等元素的宽度；`1rem = 16px` (font-size默认大小)，如果我们将`font-size`设为`100px`,则此时`1rem = 100px`。这样我们就可以将上例中的图片宽度设置为`1rem`,那么在分辨率为`640px`的设备中,`1rem`代表`100px`。

若设备分辨率为`320px`,我们可以将`font-size`设为`50px`, 图片宽度还是`1rem`保持不变，无需修改，因为此时的`1rem`就表示的是`50px`；如此通过在不同分辨率下设置font-size的默认大小,我们就可以实现在不同分辨率下的页面元素的等比例缩放效果。
